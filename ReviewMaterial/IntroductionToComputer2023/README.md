📂 C++
存放与 C++ 编程相关的课程资料，包括面向对象编程的复习材料、代码示例和练习题。

📂 思政
存放思想政治课程相关的学习资料，如课件、笔记和课后作业。

📂 高数
包含高等数学课程的课件、习题和学习资料。
